## Como Gravar

1. **Ferramenta:** Use um gravador de tela com captura de áudio (OBS Studio, Loom, ScreenPal, etc.)
2. **Resolução:** 1920×1080 (Full HD) recomendado
3. **Dados:** Popule o app com dados de exemplo antes de gravar
4. **Narração:** Siga o texto entre aspas (`> "..."`) nos roteiros — fale de forma natural e pausada
5. **Ações:** Execute as ações descritas em `[AÇÃO]` enquanto narra
6. **Edição:** Corte pausas longas para manter dentro de 2 minutos

## Convenções dos Roteiros

- `[TELA]` — O que deve estar visível na tela
- `[NARRAÇÃO]` — Texto a ser falado em voz alta
- `[AÇÃO]` — Interação a ser feita no app (cliques, preenchimento de campos)
